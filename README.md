# AI Business Agent

Production-ready AI-агент с RAG и инструментами для бизнес-процессов CRM.
Асинхронный бэкенд на FastAPI с OpenA, ChromaDB, PostgreSQL и Redis.

## Возможности

- **Чат-агент** с краткосрочной памятью (Redis)
- **RAG-поиск** по загруженным документам (PDF)
- **Инструменты CRM**:
  - Поиск клиента по email
  - Создание лида
  - Список последних сделок
- **Production-фичи**:
  - Rate limiting (Redis)
  - Retry для LLM
  - Структурированные логи (structlog)
  - Метрики запросов
  - Полностью Dockerized
